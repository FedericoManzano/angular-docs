Flex wrap - no wrap

El ancho de los elementos pueden ser fijos o variables (dependen del ancho del contenedor padre). En el caso de los ultimos mencionados cuando utilizamos flexbox el ancho se ajusta y se comprime a medida que los tamaños de pantalla son más pequeños. En el caso de los elementos con un ancho fijo, por ejemplo 300px, este ancho no debería ser modificado, pero en el caso de flexbox rompe esta recha y el ancho es reajustado para que quepa en el ancho del contenedor padre. Para que esto no suceda se utiliza una propiedad de CSS wrap.
